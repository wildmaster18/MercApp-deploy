MercApp

Aplicacion web para gestion y venta de productos

Mateo Carranza — Ingenieria en Software — Universidad Politecnica Salesiana

Resumen tecnico

MercApp es una tienda en linea compuesta por una API REST y una SPA que trabajan juntas. El backend esta construido con Node.js, Express y MongoDB, y expone endpoints para gestionar productos, categorias, autenticacion de usuarios y un chat en tiempo real con Socket.io. El frontend es una Single Page Application desarrollada con Vue 3, Vue Router y Pinia, que consume la API y presenta toda la interfaz sin recargar la pagina.

Arquitectura de despliegue

Usuario Navegador web
→
Netlify SPA Vue 3 (dist/) HTTPS + CDN
→
Railway API REST Express Helmet + Rate Limit
→
MongoDB Atlas Cluster M0 TLS activo

El frontend en Netlify se comunica con la API en Railway mediante HTTPS. CORS esta restringido al dominio de Netlify. La base de datos en Atlas acepta conexiones unicamente desde las IPs autorizadas con TLS habilitado.

Guia de ejecucion local

Requisitos

Node.js v18 o superior y MongoDB instalado localmente.

Backend

cd backend
npm install
npm run seed
npm run dev

El servidor queda disponible en http://localhost:3000. El seed crea categorias, productos de prueba y un usuario admin/admin.

Frontend

cd frontend
npm install
npm run dev

La SPA queda disponible en http://localhost:5173.
